### Block Level elements
* headings (`<h1>, <h2>, <h3>,<h4>,<h5>,<h6>`)
* div (`<div>`)
* section (`<section>`)
* footer (`<footer>`)
* article (`<article>`)
* paragraph (`<p>`)
* lists (`<ul>, <ol>`)
* nav (`<nav>`)

### Inline Level elements
* anchor (`<a>`)
* em (`<em>`)
* strong (`<strong>`)
* span (`<span>`)

### Inline Block Level elements
* img (`<img>`)
* button (`<button>`)
